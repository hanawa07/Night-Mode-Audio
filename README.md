# 🌙 Night Mode Audio (야간 모드 오디오)

맥북의 시스템 오디오를 실시간으로 처리하여 **큰 소리는 줄이고, 작은 소리는 키워주는** 야간 모드 앱입니다.
밤에 영화를 보거나 게임을 할 때, 갑자기 커지는 폭발음 등에 놀라지 않도록 도와줍니다.

## ✨ 기능
- **야간 모드 (Compressor)**: 다이내믹 레인지를 압축하여 소리 크기를 평탄하게 만듭니다.
- **볼륨 증폭 (Gain)**: 전체적인 볼륨을 확보합니다.
- **메뉴바 앱**: 상단 메뉴바(시계 옆)에서 간편하게 켜고 끌 수 있습니다.
- **시각적 피드백**: 작동 중일 때 아이콘 색상이 변합니다.

## 🚀 설치 방법 (사용자용)

이 앱은 애플 개발자 서명이 되어 있지 않으므로(Unsigned), 최초 실행 시 보안 경고가 뜰 수 있습니다.

### 1. 사전 준비 (필수)
이 앱은 가상 오디오 드라이버인 **BlackHole 2ch**가 필요합니다.
아직 없다면 터미널에서 아래 명령어로 설치하거나, [BlackHole 다운로드 페이지](https://existential.audio/blackhole/)에서 설치하세요.

```bash
brew install blackhole-2ch
```

### 2. 앱 실행하기
1. `NightModeAudio.zip`을 다운로드하고 압축을 풉니다.
2. `NightModeAudio.app`을 **응용 프로그램(Applications)** 폴더로 옮깁니다.
3. 앱을 **우클릭** 하고 **열기**를 누릅니다. (그냥 더블클릭하면 실행이 안 될 수 있습니다)
4. "확인되지 않은 개발자가..." 경고창이 뜨면 **열기**를 누르세요.
5. 마이크 권한 요청이 뜨면 **허용**해주세요. (오디오를 캡처하기 위해 필요합니다)

### 3. 문제 해결
앱이 "손상되었거나 열 수 없습니다"라고 뜬다면, 터미널을 열고 아래 명령어를 입력하세요:
```bash
xattr -cr /Applications/NightModeAudio.app
```

## 🛠️ 개발 환경
- Python 3.14
- Rumps (macOS Menu Bar)
- Sounddevice & Numpy
- PyInstaller

## ⚠️ 주의사항
- 이 버전은 **Apple Silicon (M1/M2/M3)** 맥북 전용입니다. Intel 맥북에서는 작동하지 않을 수 있습니다.
